Help

You need an image of your game disc in BIN/CUE format to use this tool. If your disc image is split across two .bin files, drop only the first .bin file (track 1) into the box below.

After downloading the randomized image, you will need to rename it to match the track listing in your .cue file, or modify your cue file to use the name of the randomized image.

Bugs

If you find a bug, open an issue on the appropriate source repository. In the description, include the seed used, a save file, and instructions for reproducing the issue. If using an emulator, include the emulator name and version.

  • Relic randomizer issues
  • Item randomizer issues
  • Frontend issues

If you're not sure where the issue belongs to, just create it in the frontend repo.
